Apunts d'Estudi: HTML Bàsic

Quan creem una pàgina web, utilitzem el llenguatge HTML per estructurar el contingut. Els elements bàsics són les etiquetes, que defineixen parts com el títol, el cos i els enllaços.

  • head: Conté informació sobre la pàgina.
  • body: Inclou tot el contingut visible.
  • a: Serveix per crear enllaços a altres pàgines.
  • img: Insereix imatges a la pàgina.

Recorda sempre tancar les etiquetes i utilitzar una estructura jeràrquica per mantenir el codi ordenat.
